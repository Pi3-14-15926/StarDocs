# 鬼谷八荒通关记录

我有一个登仙火剑流纯净档，从来没有用过任何修改器。
这是我花了好几个月，一点一点刷出来的。
虽然已经过去很久了，但我到现在仍然记忆犹新。

那是很久以前的一个暑假。
那个夏天真的很热，房间里没有空调，我也没有属于自己的电脑椅。
我就把两个小木凳叠在一起，上面放一个坐垫，一边吹着风扇，一边流着汗，一边玩着《鬼谷八荒》。

这是我接触的第一款主机（PC）游戏。
打开游戏的时候，我真的被震撼到了。
国产水墨画风，加上复杂多元的战斗系统，还有完整的修仙突破体系。
我当时就在想：“这个世界上，还有比它更好玩的游戏吗？”
所以，只要一有空，我就开始玩《鬼谷八荒》。

哈哈哈哈哈哈哈，刚开始没玩多久，还在练气期的时候就被难住了。
即使到了练气后期，永宁州还是有很多怪物根本打不过。
期间我一度想过放弃。

想着地阶筑基算了，也不一定非要天道筑基。毕竟等我筑基以后，永宁州就不会这么艰难了，没必要一直在这里找罪受。
但最后，我还是花了游戏里六年、现实里两三天的时间，凑齐了天道筑基需要的所有材料。
我总不能连游戏里这一点小小的苦难，都不愿意去解决吧。

那时候游戏还没有开发完成，还有好几个境界被封锁着，无法突破，地图应该也只开放到了云陌州。
前几个境界的突破玩法都差不多，所以我一直很期待，后面的版本还能带来什么新的玩法和乐趣。
也是因为这份期待，我买下了《鬼谷八荒》，
而后来的天骄、宗门、Mod 等版本更新，也确实没有辜负我的这份期待。
我当时真的向身边的朋友、室友、同学推荐过这款游戏。
我跟他们说：“这个游戏真的很不错，我很喜欢。”
可惜后来 Mod 和第三方平台引流这些事情，还是降低了很多玩家对它的好感。

这是我购买的第一款游戏。
也是因为它，我第一次接触了 Steam，第一次开始购买正版游戏。
以前没什么钱的时候，总觉得玩游戏嘛，只要开心就好了，正版和盗版其实无所谓。
后来稍微独立了一点，就觉得开心当然很重要，但记录也同样重要。
游戏里的游玩时长、成就，还有那些一点一点积累起来的回忆，都很重要。

感谢张三给了这个游戏一个结局。
真的很开心，能够玩到这么好的一款游戏。
